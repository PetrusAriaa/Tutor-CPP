# Notes Tutor Progdas

## Compiling
```bash
g++ -o <output_file> <input_file>.cpp
```
Contoh:
```bash
g++ -o myProgram myProgram.cpp
```
Cara menjalankan program
(windows)
```powershell
.\myProgram
```
(macOs dan Linux)
```bash
./myProgram
```

## Commands
1. Membuat folder
```bash
mkdir <nama_folder>
```
2. Berpindah folder
```bash
cd <nama_folder>
```
3. Keluar dari folder
```bash
cd ..
```
